# Variables in Python

Variables in Python are extremely simple. You do not have to define the datatype or scope of a variable.

```
name="Alice"
```

will create a variable `name` of value `"Alice"`

```
name="Alice"
name="Bob"
```

Line 2 will overwrite the name from `"Alice"` to `"Bob"`
